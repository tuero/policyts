## Algorithms
Each directory contains training and testing code, which calls into the [libpolicyts](https://github.com/tuero/libpolicyts) framework
- `bfs/`: __Best First Search__ A general search algorithm with controlled weights on the g-cost and h-cost
- `lts/`: __LevinTS__ Orseau, Laurent, et al. "Single-agent policy tree search with guarantees." Advances in Neural Information Processing Systems 31 (2018).
- `phs/`: __PHS*__ Orseau, Laurent, and Levi HS Lelis. "Policy-guided heuristic search with guarantees." Proceedings of the AAAI Conference on Artificial Intelligence. Vol. 35. No. 14. 2021.
